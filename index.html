About me

Hello! My name is Jean. Im 33 years old and im based in Brazil 🇧🇷.
Currently, im a Front-end developer (studying to become a Full-stack), and i have made multiple Landing Pages and personal web-sites as a Freelancer. I have a design background as well, so i can deal with website layouts and image manipulation.

Im looking for a full-remote job at the moment, and im sure i have a lot to add to your company.
The main stacks i use are HTML, CSS, JS, JQuery, Python and Bootstrap. I can also use NodeJS, SASS/LESS using BEM/SMACSS methodology and automatize tasks with Gulp/Grunt.

CS50P PYTHON - FINAL PROJECT

Certificate Generator is a simple python based program that allows you to make a certificate in PDF format, branding a student name and desired date to the outputed file.

The program takes a CSV(comma separated values) file containing the name and grade of a student and, if he is qualified, outputs a certificate with the student's name and current(or any other) date. This program was made as a final project for Harvard's CS50P Introduction with Python course.
